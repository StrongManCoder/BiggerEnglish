## BiggerEnglish

A study demo for iOS.

## Usage
1) You need install [CocoaPods](https://github.com/CocoaPods/CocoaPods).

2) git clone https://github.com/VichyChen/BiggerEnglish.git

3) Open the BiggerEnglish.xcworkspace.
